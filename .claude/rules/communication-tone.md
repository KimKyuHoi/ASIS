# Communication Tone — 한국어 응답은 존댓말

## Rule

Claude가 한국어로 사용자에게 응답할 때는 **항상 존댓말**을 사용한다.

- ✅ "~합니다", "~입니다", "~이에요/예요", "~하시겠어요?"
- ❌ "~해", "~지", "~다 (단정 평어)", "~야"

## 적용 범위

| 대상                                                  | 적용                                                     |
| ----------------------------------------------------- | -------------------------------------------------------- |
| 사용자에게 직접 보내는 대화 응답                      | **존댓말 강제**                                          |
| 한국어 코드 주석                                      | 평어 (`~한다`, `~된다`) 또는 명사형 — 사용자 대화가 아님 |
| 룰 문서·README 등 한국어 문서                         | 평어 서술체 (`~한다`) — 이 파일도 마찬가지               |
| 영어 텍스트 (식별자, 영어 주석, 커밋 메시지, PR 설명) | 적용 안 함                                               |

## 예

✅ "방금 lint 설정을 추가했습니다. 한 번 봐주시겠어요?"
✅ "이 부분은 의도하신 게 맞을까요?"
✅ "확인했습니다. 빌드 통과했어요."
❌ "방금 lint 설정 추가했어. 한 번 봐줄래?"
❌ "이거 의도한 거 맞아?"

## Why

대화 톤은 협업의 기본 베이스다. 일관된 존댓말 사용은 신뢰감을 주고,
긴 세션에서 톤이 흔들리지 않도록 한다.

## 자동 적용

ESLint로 강제할 수 없다 — Claude가 self-enforce 한다.
